# 홈페이지 제작 의뢰서 (고객용 작성 양식)

> 안녕하세요. 홈페이지 제작에 앞서 아래 항목을 작성해 주시면 정확한 견적과 일정을 안내드릴 수 있습니다.
> 모르는 항목은 비워두셔도 되고, 상담 시 함께 정리해 드립니다.
>
> **작성 후 이메일 또는 카카오톡으로 회신 부탁드립니다.**

---

## 📖 먼저 읽어주세요 — 홈페이지 단계별 안내

홈페이지는 필요한 기능에 따라 **5개 단계**로 나뉩니다. 본인 사업에 맞는 단계를 확인하신 후 섹션 3에서 선택해 주세요.

### 🟢 1단계 — 일반 소개 페이지
**"우리 가게/회사를 인터넷에 알리고 싶어요"**

- 회사·매장 소개, 서비스 안내, 오시는 길, 문의 전화/카톡
- 정적인 정보 페이지 (글·사진 위주)
- 모바일 자동 대응
- 💰 **30만 ~ 200만원** · ⏱ 3~7일

> **이런 분께 추천**: 동네 가게, 1인 사업자, 명함 대신 쓸 사이트가 필요한 분

---

### 🟡 2단계 — 회원 + 게시판
**"고객이 가입하고 글을 남길 수 있는 사이트가 필요해요"**

- 1단계 + 회원가입/로그인
- 공지사항·후기·문의 게시판
- 마이페이지 (내 정보 수정)
- 관리자 페이지 (회원/글 관리)
- 💰 **100만 ~ 600만원** · ⏱ 2~3주

> **이런 분께 추천**: 커뮤니티 운영, 후기 적극 받고 싶은 곳, 회원 관리가 필요한 학원·스튜디오

---

### 🟠 3단계 — 예약 시스템
**"홈페이지에서 직접 예약을 받고 싶어요"**

- 2단계 + 캘린더 예약
- 시간/날짜별 슬롯 자동 관리
- 예약 확인·취소 (정책별 환불)
- 예약 시 자동 이메일/카톡 알림
- 관리자 예약 캘린더, 매출 통계
- 💰 **400만 ~ 800만원** · ⏱ 3~5주

> **이런 분께 추천**: 원데이클래스·공방·필라테스·미용실·스튜디오·룸대여·상담예약·식당예약

---

### 🔴 4단계 — 결제 시스템
**"예약과 동시에 결제까지 받고 싶어요"**

- 3단계 + 카카오페이/토스페이먼츠/카드 결제
- 자동 영수증·환불 처리
- 정기결제(구독) 가능
- 결제 내역 관리
- 💰 **600만 ~ 1,200만원** · ⏱ 4~7주
- ⚠️ **사전 준비**: 사업자등록증, 통신판매업 신고 필요

> **이런 분께 추천**: 온라인으로 매출이 발생하는 모든 사업, 정기 결제(멤버십)가 필요한 곳

---

### 🟣 5단계 — 풀 커스텀 플랫폼
**"우리만의 시스템을 처음부터 끝까지 만들고 싶어요"**

- 4단계 + 다중 결제수단 (카카오·토스·네이버페이 동시)
- 정산 자동화, 세금계산서 발행 연동
- 쿠폰·포인트·등급 시스템
- 고급 통계·리포트 대시보드
- 외부 API 연동 (배송·SMS·재고)
- 💰 **800만 ~ 2,000만원+** · ⏱ 6~12주

> **이런 분께 추천**: 프랜차이즈, 다수 판매자가 모이는 마켓플레이스, 복잡한 비즈니스 로직이 있는 사업

---

### 💡 단계 선택 가이드

| 이런 고민이라면 | 추천 단계 |
|----------------|----------|
| "그냥 홈페이지 하나 있으면 좋겠다" | 1단계 |
| "고객이 후기 남기고 정보 받아갔으면" | 2단계 |
| "전화/DM으로 예약받기 너무 힘들다" | 3단계 |
| "예약금이라도 미리 받았으면 좋겠다" | 4단계 |
| "기존 솔루션으론 한계가 있다" | 5단계 |

> **모르겠으면 그대로 두세요.** 상담을 통해 함께 정해드립니다.
> **인프라 운영비 (월 3~20만원)** 는 단계와 무관하게 별도이며, 섹션 14에서 부담 주체를 정합니다.
