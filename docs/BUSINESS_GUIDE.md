# 홈페이지 제작 판매 — 단계별 튜토리얼 & 체크리스트

> A1STUDIO 기술 스택을 기반으로 다른 고객사 홈페이지를 제작·판매할 때 사용하는 작업 매뉴얼.
> 단계별로 체크박스를 따라가며 누락 없이 완성하는 것이 목적.

---

## 📋 공통 — 고객 인테이크 체크리스트 (모든 단계 공통)

### A. 사업자/법적 정보
- [ ] 상호명 (정식 명칭)
- [ ] 사업자등록번호
- [ ] 대표자명
- [ ] 사업장 주소 (지번/도로명 모두)
- [ ] 대표 전화번호
- [ ] 대표 이메일
- [ ] 통신판매업 신고번호 (4단계 이상 필수)
- [ ] 사업자등록증 사본 PDF
- [ ] 도메인 보유 여부 (없으면 제안)

### B. 브랜드 자산
- [ ] 로고 파일 (PNG 투명배경, SVG 권장, 최소 500x500)
- [ ] 파비콘 (없으면 로고에서 제작)
- [ ] 대표 컬러 1~2개 (HEX 코드 또는 브랜드 가이드)
- [ ] 보조 컬러 (선택)
- [ ] 폰트 선호 (없으면 Pretendard 기본)
- [ ] 대표 이미지/사진 (5~20장, 고해상도)

### C. 콘텐츠
- [ ] 메뉴 구성 (예: 홈/소개/서비스/가격/문의)
- [ ] 회사 소개글 (200~500자)
- [ ] 서비스/상품 설명 (각 항목별)
- [ ] 가격 정책
- [ ] 자주 묻는 질문 (FAQ, 5개 이상 권장)

### D. 운영 정보
- [ ] 영업시간
- [ ] 휴무일
- [ ] 위치 (네이버 지도 좌표 또는 주소)
- [ ] SNS 링크 (인스타/유튜브/블로그)
- [ ] 카카오톡 채널 (선택)

### E. 계약 사항
- [ ] 구현 범위 (1~5단계 중)
- [ ] 작업 기간 명시
- [ ] 수정 횟수 제한 (예: 시안 3회, 개발 후 2회)
- [ ] 유지보수 범위/금액
- [ ] 인프라 비용 부담 주체 (고객 or 제작자)
- [ ] 소스코드 소유권 (보통 고객사 소유)
- [ ] 책임 한계 조항 (해킹/장애)

### F. 인프라 계정 (모델 A: 고객 명의 추천)
- [ ] Vercel 계정 (고객 이메일)
- [ ] Supabase 프로젝트 (고객 이메일)
- [ ] 도메인 등록업체 계정
- [ ] SendGrid/이메일 서비스 (4단계 이상)
- [ ] PG사 가입 (4단계, 고객 사업자명의)

# 🔴 4단계 — 결제 시스템 통합

**가격: 3단계 + 200~500만 추가 · 작업기간 +1~2주**

## 4단계 작업 흐름

### Step 1. 고객 PG 가입 안내 (제작자가 대행 X, 고객 직접)
- [ ] PG사 선택 가이드 제공
  - **토스페이먼츠** (추천 기본): 빠름, 모든 결제수단
  - **카카오페이 직접**: 매출 큰 가맹점, 수수료 절감
  - **포트원**: 다중 PG 통합 관리
- [ ] 고객 사업자명의로 가입 신청
- [ ] 통신판매업 신고 확인
- [ ] 심사 통과 대기 (1~3주, 카카오페이 직접은 2~4주)

### Step 2. 심사 통과를 위한 사전 준비 (당신이 구현)

> ⚠️ **PG 심사 핵심 원칙**
> 결제 연동이 실제로 안 되어 있어도, **"상품 선택 → 배송정보 입력 → 결제하기 버튼 클릭"** 까지의 플로우가 화면상 완성되어 있어야 심사가 가능. 즉 **PG 연동 전에 UI 플로우부터 끝내야 함.**

#### 2-1. 사업자 정보 표기
- [ ] **사이트 푸터에 사업자 정보 전체 표기**
  - 상호, 대표자, 사업자번호, 통신판매업 신고번호, 주소, 전화, 이메일, 개인정보책임자
- [ ] 푸터 정보가 사업자등록증과 **완전 일치**하는지 확인

#### 2-2. 상품/서비스 페이지
- [ ] **상품 목록 페이지** (가격·이미지·간단설명)
- [ ] **상품 상세 페이지**
  - 상품명, 가격, **상품 이미지**
  - **서비스 제공기간 또는 배송기간** 명시
  - **교환 및 환불 규정** 명시
  - "구매하기" 또는 "장바구니 담기" 버튼

#### 2-3. 결제 플로우 화면 (PG 미연동 상태에서도 구현 필수)
- [ ] **장바구니 페이지** (선택, 단일상품이면 생략 가능)
- [ ] **주문/배송정보 입력 페이지**
  - 주문자 이름, 연락처, 이메일
  - **배송 주소** (실물 상품) 또는 **이용 정보** (서비스/예약)
  - 요청사항 입력란
- [ ] **결제수단 선택 화면** (카카오페이/카드 등 옵션 노출)
- [ ] **주문 최종 확인 화면**
  - 상품명, 수량, 결제금액, 배송지 요약
  - **약관 동의 체크박스** (이용약관 + 개인정보 + 결제대행 동의)
- [ ] **'결제하기' 버튼** (클릭 시 결제창 호출 또는 더미 동작이라도 OK)
- [ ] **결제 완료 페이지** (주문번호 표시)
- [ ] **주문 내역 조회 페이지** (마이페이지)

#### 2-5. 약관 페이지

- [ ] **결제대행 이용약관** (PG사 가입 후 받는 표준 문구)
- [ ] 회원가입 시 약관 동의 체크박스

- [ ] **사이트 URL** (실제 접속 가능)
- [ ] **사업자등록증 사본** PDF
- [ ] **로그인 계정** (비회원 결제 불가 시)
