# 📱 예약 확정 알림 메시지 설정 가이드

예약이 확정되면 고객에게 자동으로 SMS/알림톡을 발송하는 기능입니다.

## 🎯 기능

- ✅ 예약 확정 시 자동 메시지 발송
- ✅ 연습실/파티룸 각각 다른 템플릿
- ✅ 예약 정보, 입실 안내, 시설 이용 안내 포함
- ✅ 메시지 발송 로그 자동 저장
- ✅ 발송 실패 시에도 예약은 정상 진행

## 🔧 설정 방법

### 1️⃣ COOLSMS 가입 및 API 키 발급

1. [COOLSMS 웹사이트](https://www.coolsms.co.kr/) 접속
2. 회원가입 및 로그인
3. **포인트 충전** (건당 약 20원, 테스트는 무료)
4. **API 키 발급**:
   - 콘솔 > API Keys > 새 API Key 생성
   - API Key와 API Secret 복사

5. **발신번호 등록**:
   - 콘솔 > 발신번호 관리 > 발신번호 등록
   - 본인 명의 휴대폰 번호 등록 및 인증

### 2️⃣ 환경변수 설정

`.env.local` 파일에 다음 내용을 추가하세요:

```bash
# ── SMS/알림톡 (COOLSMS) ──
COOLSMS_API_KEY="발급받은_API_KEY"
COOLSMS_API_SECRET="발급받은_API_SECRET"
COOLSMS_FROM_NUMBER="01012345678"  # 등록한 발신번호 (- 제외)
```

### 3️⃣ 테스트

1. 애플리케이션 재시작
2. 예약 생성
3. 콘솔 로그 확인:
   ```
   [SMS] 발송 성공: { groupId: '...' }
   [예약] 확정 메시지 발송 성공: ...
   ```

4. COOLSMS 콘솔에서 발송 내역 확인

## 📊 메시지 발송 로그 확인

Supabase 대시보드에서 `message_logs` 테이블을 조회하여 발송 내역을 확인할 수 있습니다:

```sql
SELECT 
  created_at,
  phone_number,
  message_type,
  status,
  error_message
FROM message_logs
ORDER BY created_at DESC
LIMIT 50;
```

## 🎨 메시지 템플릿 수정

메시지 내용을 수정하려면 다음 파일을 편집하세요:

```
src/lib/message-templates.ts
```

함수:
- `getPracticeRoomConfirmMessage()` - 연습실 예약 확정
- `getPartyRoomConfirmMessage()` - 파티룸 예약 확정
- `getCancelMessage()` - 예약 취소 (미구현)

## 🚨 문제 해결

### 메시지가 발송되지 않아요
1. 환경변수가 올바르게 설정되었는지 확인
2. COOLSMS 포인트 잔액 확인
3. 발신번호가 등록되었는지 확인
4. 콘솔 로그에서 에러 메시지 확인

### 메시지는 발송되는데 받지 못해요
1. 수신자 번호가 올바른지 확인
2. 스팸 차단 앱 확인
3. COOLSMS 콘솔에서 발송 상태 확인
