# 카카오 로그인 개인정보 동의 항목 설정 가이드

## 개요
카카오 로그인 심사를 통과하기 위해 **필수 동의 항목**과 **개인정보 처리방침** 설정이 필요합니다.

## 📋 필수 수집 항목
- **이름** (profile_nickname)
- **출생 연도** (birthyear) - 성인 인증용
- **카카오계정(전화번호)** (phone_number) - 예약 안내 메시지 발송용

## 🔧 카카오 개발자 콘솔 설정 방법

### 1. 카카오 개발자 콘솔 접속
1. https://developers.kakao.com 접속
2. 애플리케이션 선택
3. **[제품 설정] > [카카오 로그인]** 메뉴로 이동

### 2. 동의 항목 설정
**[제품 설정] > [카카오 로그인] > [동의 항목]**에서 다음과 같이 설정:

#### 필수 동의 항목 (3개)
| 항목명 | 설정 | 설명 |
|--------|------|------|
| 닉네임 (profile_nickname) | **필수 동의** | 회원 식별용 |
| 출생 연도 (birthyear) | **필수 동의** | 성인 인증 (만 19세 이상 확인) |
| 카카오계정(전화번호) (phone_number) | **필수 동의** | 예약 확인 및 안내 메시지 발송 |

#### ⚠️ 중요: 개인정보 처리방침 설정
- **개인정보 처리방침** 항목을 **필수 동의**로 설정
- 개인정보 처리방침 URL 입력:
  ```
  https://a1-studio.vercel.app/privacy
  ```
  (또는 배포된 실제 도메인 사용)

### 3. 수집 목적 명시
각 항목의 "수집 목적"란에 다음과 같이 입력:

**닉네임:**
```
회원 식별 및 인증
```

**출생 연도:**
```
성인 인증 (만 19세 이상 확인) - 주류 반입 가능 파티룸 운영
```

**카카오계정(전화번호):**
```
예약 확인 및 안내 메시지(SMS/알림톡) 발송
```

### 4. 비즈 앱 전환 및 심사 신청
1. **[앱 설정] > [일반]**에서 **비즈 앱 전환** 신청
2. 사업자등록증 업로드
3. **[제품 설정] > [카카오 로그인] > [검수 신청]**
4. 필수 동의 항목 심사 신청

## ✅ 코드 수정 완료 항목

### 1. 카카오 로그인 Scope 설정
**파일:** `src/lib/auth-client.tsx`

```typescript
// 카카오 로그인 시 필수 동의 항목 설정
const scopes = provider === "kakao" 
  ? "profile_nickname,birthyear,phone_number" // 필수: 이름, 출생 연도, 전화번호
  : undefined;
```

### 2. 로그인 페이지 동의 설명 업데이트
**파일:** `src/app/login/page.tsx`

- 필수 수집 항목: 이름, 출생 연도, 카카오계정(전화번호)
- 이용 목적: 회원 식별, 성인 인증, 예약 안내 메시지 발송
- 개인정보 처리방침 동의 필수 체크

### 3. 개인정보 처리방침 업데이트
**파일:** `src/app/privacy/page.tsx`

제3조(개인정보 이용 목적)에 다음 추가:
- 성인 인증 (만 19세 이상 확인) - 주류 반입 가능 파티룸 운영
- 예약 확인 및 안내 메시지(SMS/알림톡) 발송

## 📝 심사 시 참고사항

### 반려 사유 대응
| 반려 사유 | 해결 방법 |
|----------|----------|
| 개인정보 처리방침이 선택 동의로 설정됨 | 필수 동의로 변경 |
| 수집 목적이 불명확함 | 구체적인 사용 목적 명시 (성인 인증, 예약 안내 등) |
| 필수 항목과 처리방침 불일치 | 코드와 콘솔 설정을 일치시킴 |

### 심사 통과 체크리스트
- [ ] 개인정보 처리방침 URL이 올바르게 설정됨
- [ ] 개인정보 처리방침 동의가 **필수**로 설정됨
- [ ] 3개 필수 항목이 모두 **필수 동의**로 설정됨
- [ ] 각 항목의 수집 목적이 명확히 기재됨
- [ ] 처리방침 페이지(`/privacy`)에 수집 항목과 목적이 명시됨
- [ ] 로그인 페이지에서 동의 체크박스가 필수로 작동함
- [ ] 비즈 앱 전환 완료
