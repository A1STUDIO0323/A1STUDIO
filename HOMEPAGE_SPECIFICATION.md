# A1 STUDIO 홈페이지 전체 구성 명세서

**프로젝트명**: A1 STUDIO 웹사이트

## 📐 홈페이지 섹션 구조

### 1. **HeroScroll** (히어로 스크롤 섹션)
- **높이**: 300vh (스크롤로 애니메이션 진행)
- **특징**: 
  - 스크롤 진행도에 따라 변화하는 인터랙티브 히어로
  - 다크 배경(#0f0e0d) → 라이트 배경(#F7F3EB) 전환
  - 앰비언트 글로우 효과 (블러된 원형 그라데이션)
  - 3단계 애니메이션 시퀀스

#### 애니메이션 시퀀스:
1. **초기 (0~0.30)**
   - A1 STUDIO 로고 표시 (중앙)
   - 스크롤 인디케이터 (하단, 애니메이션)
   - 로고가 서서히 페이드아웃 + 축소 + 상단으로 이동

2. **중간 (0.16~0.46)**
   - 스튜디오 정보 오버레이 등장
   - 어두운 글래스 패널 안에 표시
   - 주소, 지하철역 정보, 구비 비품 정보
   - 페이드 인/아웃으로 잠깐 표시

3. **후반 (0.24~0.62)**
   - 배경이 크림색으로 전환
   - 하단에서 미리보기 카드 상승
   - "One Space · Four Uses" 4칸 그리드
   - 보컬/댄스/연기/뮤지컬 용도 아이콘 표시

#### 기술 구현:
```typescript
useScroll() // motion/react
useTransform() // 스크롤 진행도 → CSS 값 변환
useReducedMotion() // 접근성 고려
```

## 🎭 애니메이션 특징

### 스크롤 기반 애니메이션
- **HeroScroll**: useScroll + useTransform
- 스크롤 진행도에 따라 실시간 변환
- 부드러운 이징 함수 사용

### 뷰포트 기반 애니메이션
- **whileInView**: 화면에 보일 때 애니메이션
- **once: true**: 한 번만 실행
- **순차 등장**: delay를 활용한 stagger 효과

### 반복 애니메이션
- 스크롤 인디케이터: y축 무한 반복
- CTA 화살표: y축 부드러운 바운스

### 접근성
- **useReducedMotion()**: 사용자 설정 감지
- 애니메이션 비활성화 옵션 제공

## 📱 반응형 디자인

### 브레이크포인트
- **모바일**: < 640px
- **sm**: ≥ 640px (태블릿)
- **md**: ≥ 768px
- **lg**: ≥ 1024px (데스크톱)

### 주요 반응형 패턴
1. **그리드**:
   - 모바일: 1~2열
   - 태블릿: 2~3열
   - 데스크톱: 3~4열

2. **타이포그래피**:
   - 모바일: text-sm, text-base
   - 데스크톱: text-lg, text-xl

3. **패딩/마진**:
   - 모바일: p-4, py-14
   - 데스크톱: p-6, py-20
